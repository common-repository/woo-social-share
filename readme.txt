== Description ==

**Woo Social Share**

Woo Social Share Lite Plugin is a state-of-the-art next generation plugin that allows all online shops, ecommerce stores and related sites to integrate and share their every new product on Facebook. The users just need to add Facebook API key and secret key of Facebook application and its done.
Woo Social Share Lite Plugin functions automatically as it instantly shares the product on Facebook once it is posted on the site. The functionality of Woo Social Share Lite Plugin is simpler than ever and that makes it one of the must-have plugins for every woocommerce site. Just install and leave the rest on this plugin.

= Plugin Features =
* Auto Share Product on Facebook 
* Secure Connect With Facebook Account
So go ahead guys and install it now!


== Installation ==
1. Upload `woo-social-share` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Activate the plugin via the 'Plugins' menu in WordPress, you'll automatically redirected to plugin admin panel.
3. Plugin settings are located in 'Social Share'.
